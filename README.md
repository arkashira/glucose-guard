# Glucose Guard Simulator

A Python project to simulate glucose sensor data for testing and development purposes.

## Features

* Generates realistic sensor data
* Allows for customizable data parameters
* Integrates with existing AI-powered analytics

## Usage

1. Install the project using `poetry install`
2. Run the simulator using `python -m glucose_guard`
3. Customize data parameters using the `customize_data_parameters` method
4. Integrate with analytics using the `integrate_with_analytics` method

## Testing

1. Run tests using `pytest`
