# Glucose Guard

AI Trend Alerts for Glucose Levels

## Introduction

Glucose Guard is a Python project that provides AI trend alerts for glucose levels. It predicts the trend of glucose readings and sends alerts when a potential hypo- or hyper-glycemia event is detected.

## Usage

1. Install the project using `poetry install`.
2. Run the tests using `pytest`.
3. Use the `GlucoseGuard` class to predict trends and send alerts.
